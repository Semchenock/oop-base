# Bank Project

Учебный проект на Python, моделирующий банковскую систему с несколькими типами счетов и инвестиционными активами.

Проект демонстрирует использование:

* ООП
* наследования
* инкапсуляции
* пользовательских исключений
* работы с перечислениями (`Enum`)

# Типы счетов

## BankAccount

Базовый банковский счет.

Функциональность:

* уникальный ID счета
* владелец счета
* баланс
* валюта
* статус счета (`ACTIVE`, `FROZEN`, `CLOSED`)

Основные операции:

* `deposit()` — пополнение счета
* `withdraw()` — снятие средств
* `get_account_info()` — получение информации о счете

---

## SavingsAccount

Сберегательный счет с минимальным балансом и процентами.

Дополнительные параметры:

* `min_balance` — минимальный остаток
* `monthly_interest` — ежемесячная процентная ставка

Дополнительный метод:

* `apply_monthly_interest()` — начисление процентов на баланс

---

## PremiumAccount

Премиальный счет с комиссией за снятие и возможностью овердрафта.

Дополнительные параметры:

* `fee` — комиссия за снятие средств
* `overdraft_limit` — допустимый лимит отрицательного баланса

Особенности:

* при снятии средств автоматически применяется комиссия
* баланс может уходить в минус в пределах `overdraft_limit`

---

## InvestmentAccount

Инвестиционный счет, который хранит виртуальные активы.

Содержит список объектов `Asset`.

Дополнительный метод:

* `project_yearly_growth()` — расчет ожидаемой годовой доходности портфеля

---

# Инвестиционные активы

## Asset

Описывает инвестиционный инструмент.

Параметры:

* `asset_type` — тип актива
* `name` — название
* `price` — стоимость
* `yearly_interest` — ожидаемая годовая доходность

Типы активов (`AssetType`):

* `STOCK` — акции
* `BOND` — облигации
* `ETF` — биржевые фонды

# Исключения

Проект содержит собственные классы ошибок:

* `InvalidOperationError` — неверная операция (например отрицательная сумма)
* `InsufficientFundsError` — недостаточно средств
* `AccountFrozenError` — счет заморожен
* `AccountClosedError` — счет закрыт

# Пример использования

```python

    balance=1000,
```
